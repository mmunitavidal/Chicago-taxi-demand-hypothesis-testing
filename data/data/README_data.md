# Fuente de datos

Este proyecto utiliza un **dataset educativo proporcionado por TripleTen (bootcamp Data Analyst)**.

## Dataset
- Fuente: TripleTen (material del bootcamp)
- Contenido: tablas/archivos del caso (usuarios, planes y uso mensual, según corresponda)

## Cómo reproducir
1. Descargar los archivos desde la plataforma del bootcamp TripleTen.
2. Guardarlos dentro de la carpeta `/data/` manteniendo los nombres originales.
3. Ejecutar el notebook ubicado en `/notebooks/`.

## Nota
El dataset no se incluye en el repositorio por tamaño/licencia y por pertenecer a material del programa.
